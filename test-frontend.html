SZork Frontend Test

The frontend is now running with the fixes applied.

To test:

  1. Open the browser developer console (F12)
  2. Navigate to: http://localhost:3091/
  3. Start a new adventure
  4. Check if:
    • Images are displayed after generation
    • Background music plays after generation

What was fixed:

  • Image issue: The composable was setting message.image but GameView.vue was looking for msg.imageUrl. Fixed by checking for msg.image first.
  • Music issue: The composable was calling its own stub playBackgroundMusic function instead of the real one in GameView.vue. Fixed by:
    • Adding callback support in the composable
    • Setting the callbacks in GameView.vue's onMounted
